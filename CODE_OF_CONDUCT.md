# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Short version: be respectful, assume good faith, no harassment of any kind,
critique code not people. Maintainers may edit/remove contributions and ban
contributors who violate this.

Report issues privately via GitHub to the maintainer
([@4arkeinlvy](https://github.com/4arkeinlvy)). Reports are handled
confidentially.
